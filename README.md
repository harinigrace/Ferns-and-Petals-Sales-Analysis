# Ferns-and-Petals-Sales-Analysis
This project analyzes the sales performance of Ferns and Petals, an online
gifting company. The objective of the project is to identify sales trends,
customer purchasing patterns, product performance, and revenue insights
through an interactive Excel dashboard.
The project was developed using Microsoft Excel and involved data
cleaning, transformation, analysis, and visualization to support business
decision-making.
